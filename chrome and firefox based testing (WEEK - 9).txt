week - 9 Build and Deploy Selenium Grid for Chrome and Firefox

Go to AWS EC2 Console
 Launch a new instance with the following settings:
o Name: SeleniumGridServer
o AMI: Ubuntu Server 22.04 LTS
o Instance Type: t2.micro (for demo) or t3.small (recommended)
o Key Pair: Create or select existing
o Security Group Rules:
1. SSH (22)
2. Custom TCP – Port 4444 (Anywhere)


Connect using SSH
------
sudo apt update 
sudo apt install -y docker.io 
sudo systemctl start docker
sudo systemctl enable docker
sudo curl -L"https://github.com/docker/compose/releases/download/v2.17.3/docker-compose-$(uname-s)-$(uname-m)"-o /usr/local/bin/docker-compose
sudo chmod +x /usr/local/bin/docker-compose
docker--version 
docker-compose--version
mkdir selenium-grid 
cd selenium-grid 
nano docker-compose.yml
   

paste it 

version:"3.8" 
services: 
selenium-hub: 
image: selenium/hub:4.18.1 
container_name: selenium-hub 
ports: -"4444:4444" 
chrome: image: selenium/node-chrome:4.18.1 
container_name: chrome-node 
shm_size: 2gb 
depends_on: 
- selenium-hub 
environment: 
- SE_EVENT_BUS_HOST=selenium-hub 
- SE_EVENT_BUS_PUBLISH_PORT=4442 
- SE_EVENT_BUS_SUBSCRIBE_PORT=4443 
firefox: 
image: selenium/node-firefox:4.18.1 
container_name: firefox-node 
shm_size: 2gb 
depends_on: 
- selenium-hub 
environment: 
- SE_EVENT_BUS_HOST=selenium-hub 
- SE_EVENT_BUS_PUBLISH_PORT=4442 
- SE_EVENT_BUS_SUBSCRIBE_PORT=4443



sudo docker-compose down 
sudo docker system prune -af 
sudo docker-compose up -d
docker ps
<EC2-PUBLIC-IP>:4444/ui
